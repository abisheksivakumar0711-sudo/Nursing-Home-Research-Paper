# Digital Healthcare Governance System

## Overview
This project presents a conceptual ERP-inspired digital platform for nursing home planning and healthcare governance in Germany.

The research applies Design Science Research (DSR) methodology together with TOGAF enterprise architecture principles to address fragmentation, inefficient stakeholder coordination, and legacy-system dependency in public healthcare planning.

## Research Focus
The project explores:
- Enterprise Architecture (TOGAF ADM)
- ERP and Business Intelligence integration
- Digital governance systems
- Public sector digital transformation
- Role-Based Access Control (RBAC)
- Healthcare process digitalization
- System interoperability and scalability

## Key Contributions
- Designed modular ERP-inspired architecture
- Developed UML component and layered architecture diagrams
- Proposed BI-supported evaluation workflows
- Integrated governance and compliance requirements
- Applied user-centric and sensemaking-based UI concepts
